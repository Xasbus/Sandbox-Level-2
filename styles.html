Background Image

          How to add background image.
            - background-image: url(dragon-png.jpg);
            - Add the file name or url for the image. png are best for backgrounds.
            - background-image: linear-gradient (yellow, blue, red)
            - Add to right, to left, to bottom right, to bottom left  ~Changes direction of colors~
            - Changing direction can use degrees too
                90deg
                125deg
                270deg
                225deg
                
          Resizing the background-image
            - create a class using, id
            - .mySize {
              background-size: 20%;
            }

          Repeating a background
            1. no repeat
            2. repeat
              - by default it repeats
              - use background-repeat: no repeat;
              [ create a class called no-repeat ]
               * Example: .no-repeat {
                  background-repeat: no repeat;
                  }
              - use background-repeat: space;
              [ repeats the background but doesn't allow it to be cut off ]
              - first value is horizontal (x axis)
              - second value is vertical (y axis)
              - values are seperated but space only
                * Example: background-repeat: space no repeat
                *                           (x-axis)  (y-axis)
          Background position
            1. center
            2. top
            3. bottom
            4. right
            5. left
              * Example: background-position: center;
            6. off-center
              - uses two values using (x axis) and (y axis) indivually
              - 1st value is (x)
              - 2nd value is (y)
              * Example: background-posistion: 20% 30%;

Opacity


          Using <opacity> you can set an element's transparent
            - 0 is invisible
            - 1 is fully visible
            - rgb (x,x,x, 0.5)
        
          display: none vs opacity:0
            * display: none
              - makes the element invisible
              - removes the element from the browser window
            * opacity:0
              - makes the element invisible
              - element still takes up space in the browser

Specificity


          Order of styling priority
            1. inline
            2. id
            3. class
            4. element
          Styles with higher priority overrides lower.
          !important is set to the highest priority and overrides them all.
          !important also goes by loading order when dealing with more than one. Last one gets applied.
        Examples below:
EXAMPLE ELEMENT: Hello World is on element tag and is the most general. Classes are a step more specific. ID are another step more specific. And inline is the most specific.
EXAMPLE CLASS: Hello World is on element tag and is the most general. Classes are a step more specific. ID are another step more specific. And inline is the most specific.
EXAMPLE ID: Hello World is on element tag and is the most general. Classes are a step more specific. ID are another step more specific. And inline is the most specific.
EXAMPLE INLINE: Hello World is on element tag and is the most general. Classes are a step more specific. ID are another step more specific. And inline is the most specific.

Padding Shorthand Property



          Padding with 1 value sets all sides equally
            Eaxmple: padding: 10px

          Padding with 4 values
            1. top
            2. right
            3. bottom
            4. left
              Example: padding: 5px, 7px, 8px, 9px
              *Padding gets applied in a clockwise direction.*
          
          Padding with 2 values
            1. top & bottom
            2.left & right
              Example: padding 10px, 20px
          
          Padding with 3 values
            1. top
            2. left and right
            3. bottom
              Example: padding 10px, 5px, 15px
