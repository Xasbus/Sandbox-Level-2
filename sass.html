        • Use colon to store a value

How to use SASS variables to remember a file name

        $dragonPic: url(dragon-png.jpg);

SASS variable to remember horizontal repeat

        
        $horizontalRepeat: space no-repeat;
          .horizontalRepeat {
          background-repeat: $horizontalRepeat;
          }

SASS variable for veritcal repeat

        
        $verticalRepeat: no-repeat repeat;
          .verticalRepeat {
          background-repeat: $verticalRepeat;
        }

SASS variable for stretching backgrounds

        
        $stretch: 25% 100%;
        .stretch {
          background-size: $stretch;
        }

SASS variable for using gradient stops

        
        $darkGradient: linear-gradient(90deg, gray, black 10% 90%, gray);
        $lightGradient: linear-gradient(90deg, gray, white 10% 90%, gray);
        .gradient {
          background-Image: $lightGradient;
          }

Adding Fonts

        1. Download the font - used dafont.com
        2. If the font is .zip - move the font to root folder
        3. Use @font-face to add the font
        4. Use font-family to name the font
        5. Use src to link the font file
        6. Use the font.
        
          @font-face {
            font-family: "Bleeding Cowboys";
            src: url("Bleeding_Cowboys.ttf");
        }

Single Line Mixin

        
        @mixin redText {
          color: red;
        }

        - Mixin are like variables
        - Single line mixins store a property and value
        - Type @mixin to define/declare a mixin variable
        - Type @include to use the mixin
        - Mixins can substitute for utility class so elements can be described by their names
