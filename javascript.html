Callback Functions

      function myFunction (callbackFunction) {
            callbackFunction();
            }
          function activator() {
            myFunction(greeting);
            }
         activator();
          
          
        Is a function that is passed as a parameter.
        A function can be passed to another function.
        Function can run that function.

Set Time Out

         setTimeout(greeting, 5000);
         
        A fucntion that calls a function after a time.
        It accepts a function and delays it in miliseconds (1000 = 1s)

Asynchronous

          
        function asynchronous() {
          setTimeout(greeting, 5000);
          display("Waitng for the greeting.....");
        }
          
          Able to run code while waiting for other functions to run.
          Like multitasking.
